# Learning_GL

- main.cpp 是形成了箱子的地图，每一个箱子都贴图了，有环境光照，有鼠标/键盘
- voxol_main ，去除了环境光照，使用相机纹理

## 编译方法

- cmke 编译，可以直接使用 vscode 集成

```shell
cd ..
rm -rf build
mkdir -p build
cd build
cmake ..
make
./OpenGL
```

## env

- glfw 直接编译
- glew 使用了系统的库
- 下载 glad C++ opengl4.6 core  https://glad.dav1d.de/
